Method

Using TCP Socket APIs
Implement C/S model: 1. server listens to a given port; 2. client initiates a TCP connection to the server; 3. client send a request to download a file/text; 4. server respond with the file/text; 5. the client save the file to local directory. Repeat the before request-respond (3 to 5) until 'esc' is given, then client tear down the TCP connection. Detailed relationship shown as:
