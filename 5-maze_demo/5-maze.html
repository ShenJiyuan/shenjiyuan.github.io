Smart Vehicle Tracking and Identifying Control System

Jiyuan Shen, Xuemin Yang, Cheng Chang and Yuxuan Song

Goal

Given a mini-car and references included, let the mini-car automatically solve maze games.
Specification: Single chip with bluetooth module. There are specifically two side: for computer side, it should process the maze image and produces step strategies; for single chip side, it should follow the computer instruction and move correspondingly.

My Work

♣ Co-developed a quadrotor control system based on A.R.Drone platform.
♣ Designed line-tracking and color-identifying algorithms.
♣ Implemented video streaming process and bench- marked image pattern recognition problem on built-in camera.

Method

Step 1: Single-chip side: Tear down the steering engine and only use wheels. Embed motor moving strategies for control commands.
Exaplanation: Normally the wheels control forwarding while steering engine controls turning. But if we implement it only through wheels, we can finnaly achieve spinning around action as well. It is very helpful for mini-car to solve maze games.

Step 2: Computer side: Real-time Image Processing. Inverse the color of maze to get fined path of capable ways instead of walls. Then apply A* Search Algorithm in each iteration to determine which of its partial paths to expand into one or more longer paths.
Explanation: Notice the special characteristics of given mazes, that is, color for wall and white for path.
