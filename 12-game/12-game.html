Air Attack Game Design

Jiyuan Shen

My Work

♣ Developed air attack game with real-time record.
♣ Designed human-computer interaction.
Though documents are in Chinese, the game is easy to play without help.

Method

Packages The project is implemented by Python. We import pygame and random packages for usage.
Classes
Class Bullet: set up location and each movement distance by using self.poslist and self.delta.
Class Player: set up player's flight location by using getpos function. Then let bullet location correspons to the player's flight location and set up its movement function 'move'. At the same time, check if the flight is within board.
Class Enemy: randomly generate scales of enemies by using random.randint. Calculate real-time scores by using self.score.
