Neural Network Tool

Jiyuan Shen

Goal

The tool is utilized for neural network related experiments.
Specification: Included are C++ source codes containing fully-connected layers, convolutinoal layers, output layers and maxpooling layers. There is also a mnist parser included for mnist related tests. Codes are from scratch and simple, please feel free to check it.

Content

Based on Boost Libraries.
1. Activation Activation functions can be refered in 'activation.h' and 'activation.cpp'. Sigmoid and Tanh are included already.
2. Back Propagation In fully-connected layers, convolutinal layers, output layers, back propagation is utilized for training.
3. Retraining Retraining neural networks by given a trained weight file.
4. Redundancy Redundant rows are available for weight matrix mapping.
Detailed implementation is explicit, and please refer to the source codes directly.
