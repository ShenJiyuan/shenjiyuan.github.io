Set up Github SSH Key on Ubuntu

Jan 24, 2016

This blog is about how to set up github SSH key on ubuntu.

ssh-keygen -C 'jiyuansmile@gmail.com' -t rsa

Copy the key from ~/.ssh/id_rsa.pub, and add the key in the github settings.

Test via

ssh -T git@github.com

Now you can test whether you can clone repos without password. If not, check /etc/ssh/sshd_config, change PasswordAuthentication to be Yes, and restart ssh service.
