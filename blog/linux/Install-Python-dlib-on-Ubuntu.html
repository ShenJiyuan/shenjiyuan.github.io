Install dlib with Python on Ubuntu

Mar 12, 2016

This blog is about how to install dlib library with python on ubuntu.

1. Make sure boost has been installed successfully. Download package boost.org, unpack it and go to the directory.

./bootstrap.sh
./b2 install

2. Make sure other dependencies are done.

sudo apt-get install libboost-python-dev cmake

3. Install dlib with python.

sudo python setup.py intall

4. Test via

python
>> import dlib
>>
