Patent Classification Modeling based on Machine Learning

Jiyuan Shen, Junwei Yan

Goal

Use simple NN strategies to solve patent classification; experiments on BCMI datastes.

My Work

♣ Devised the classification algorithm mainly in learning kernel for massive patent classification.
♣ Evaluated with BCMI dataset and achieved results with high accuracy.
♣ Used multiple passing interface to achieve parallelism, and implemented parallel training (reduced time 2212s to 80s).

Method

Refer to the report for implementations.
Step 1: Liblinear: use regular single thread to learn and experiment. Notice
Step 2: MinMax Method: ramdonly split the original problem to subproblems in scale, each subproblem is learned by using Liblinear, then MinMax the sub-results. Multiple threads are utilized.
Step 3: Based on Pre-knowledge and MinMax: instead of randomly split the original problem, here we use pre-knowledge to split it to subproblems, then each subproblem is laerned by Liblinear and Minmax their results. Multiple threads are utilized as well.
Step 4: Calculate the F1 value of this three classifiers, draw out their ROC curve, and make a comparison.
Step 5: Implement a classifier, such as MLP or SVM polynomial kernal, and make comparison with LibLinear.
